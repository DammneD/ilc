# UI for ILC Registry

## Quick start in dev mode

```bash
$ npm install
$ npm start
```

And then browse to [http://localhost:8080/](http://localhost:8080/).

The default credentials are: **root / pwd**

## How to build/run

```bash
$ npm run build
$ npm run serve
```
